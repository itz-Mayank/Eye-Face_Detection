Secure Face Authentication

Using Edge AI and Model Protection for UIDAI

UIDAI has implemented a secure face authentication solution for mobile and desktop browsers, leveraging AI on the edge. The system uses optimized ML models (5 MB, 500 ms inference time) with obfuscation and cryptography for protection. It calculates a confidence score, requiring a minimum of 95% for success. Users perform tasks like eye calibration, blinking, and smiling, with up to two attempts for verification. After two failed attempts, access is locked for the day. The solution supports high-volume transactions and offers scalable, real-time, and user-friendly authentication.

Position Yourself:

Perform the Challenge:

  • Trace red dots, say the displayed words aloud, and blink to continue.
